表单修饰符

  • .lazy
  • .number
  • .trim

.lazy

  • 用户输入内容时不做绑定数据的更新处理，在控件onChange时间中更新绑定的变量

##.number

  • 将用户输入的内容转换为数值类型，如果用户输入非数值的时候，返回NaN

.trim

  • 自动去掉用户输入内容两端的空格
